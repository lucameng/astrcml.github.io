message_table module¶

NOTE:¶

This script is used to store field names of adas_v2 message. The sequense of keys in each list are fixed and it’s a one-to-one correspondence with logic defined in data_processor.py, so that, one shall not change the sequence of keys, while name editing is recommended so as to apply to name change in the underlying protocol.
